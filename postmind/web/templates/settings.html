Use postmind from Claude Code

Connect Claude Code CLI or Claude Desktop to your inbox — postmind is already running an MCP server at http://127.0.0.1:8484/mcp/sse.

Claude Code CLI

Run this once — postmind will be available in every future Claude Code session while this server is running.

claude mcp add postmind --transport sse http://127.0.0.1:8484/mcp/sse -s user

Claude Desktop

Add to ~/Library/Application Support/Claude/claude_desktop_config.json and restart Claude Desktop.

{
  "mcpServers": {
    "postmind": {
      "url": "http://127.0.0.1:8484/mcp/sse"
    }
  }
}
